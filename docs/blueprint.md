# **App Name**: DoView Admin Dashboard

## Core Features:

- Dashboard Overview: Display key metrics and overviews of the business at a glance. Including charts powered by AI to identify trends
- Collapsible Sidebar: Implement a responsive sidebar that collapses to show only icons for a cleaner, more focused user experience.
- Itinerary Management: Allow creation, viewing, and confirmation of travel itineraries.
- Account Management: Enable management of different user accounts, including agents, guides, and staff.
- Daily Moment Tracking: Track and display daily activity moments for operational insights.
- Performance Insights: AI tool to highlight key performance metrics, such as top-rated agents and booking trends, displayed within dedicated 'Star Performers' and metrics cards to help admins optimize resource allocation.

## Style Guidelines:

- Primary color: Vivid rose (#F72585) to reflect the exciting nature of travel.
- Background color: Very light lavender (#F2EDF8) to give a clean, calming feel to the main content area.
- Accent color: Deep purple (#4361EE) to complement the rose and add depth, used for interactive elements and key indicators.
- Font pairing: 'Space Grotesk' for headings, providing a tech-forward feel, combined with 'Inter' for body text, ensuring readability.
- Use line icons, similar to Feather or Heroicons, for a clean and modern look.
- Employ a responsive layout with a collapsible sidebar and a centered main content area, using flexbox and CSS grid to ensure proper alignment and spacing.
- Incorporate subtle transitions and animations for a polished and engaging user experience, particularly when the sidebar is toggled.
